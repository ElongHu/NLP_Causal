## vocab
    作为数据部分，有很多细节。
    具体的特征与设计如下
    模型需要的是数字而不是字符串所以词表需要构建一个编号到词词到编号的双向映射
    并非所有的token都要参与训练，一般情况可以取频率高的部分训练
    频率越高的越可能被频繁用到这个时候将频率高的数据放在一块就可以充分利用这个局部性原理
    最后，词表是一个类。
